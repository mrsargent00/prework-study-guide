HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • h1 refers to the level of heading a given text block represents
  • p element represents a paragrah or a block of text
  • img element contains information about images that are displayed on a webpage

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Always end a line of code with a semi-colon, or the code will not register
  • Link a CSS code by adding link rel ="stylesheet" href="./assets/style.css" in the main area for code

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • feature/starter-code: a name of a new branch
  • git pull origin main: makes local branch in sync with the base branch in GitHub

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Use console.log to output a message to the web console by adding an arguement
  • Conditional statements and loops control the order in which code is executed, or if it is even executed at al
